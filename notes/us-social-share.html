美国社交 App 分享机制拆解

拆解 Facebook、Instagram、Messenger、X、WhatsApp、TikTok、Pinterest 等主流平台的分享入口、 卡片样式、落地路径与能力差异，为后续产品内分享模块设计提供结构化参考。

概要说明

这份笔记聚焦于美国主流社交 App 的“分享链路”，包括： 分享入口在产品中的位置、分享流程中每一步的用户动作、 Web / App 不同端的行为差异，以及分享卡片（预览图、标题、描述、Deep Link）的生成逻辑。

同时记录各平台在 URL 解析、OG 标签支持、schema / intent、跳转落地页以及 API / SDK 限制方面的差异，用来反推「在自己的产品里，怎样设计一套兼容性较好的分享模块」。
